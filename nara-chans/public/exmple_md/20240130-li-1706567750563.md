# markdownの書き方 チートシート

## 見出し
### 書き方  
``` md
# h1  
## h2  
### h3  
#### h4  
##### h5  
###### h6  
```
**(例)**
# h1

## リスト(箇条書き)
「-」「+」「*」のいずれかが使える
``` md
- A1
  - B1
    - C1
    - C2
- A2
```
```
+ A1
  + B1
    + C1
    + C2
+ A2
```
```
* A1
  * B1
    * C1
    * C2
* A2
```
**(例)**
- 食べたい物
  - 揚げ物
    - 唐揚げ
    - エビフライ
- 行きたい場所

## リスト(番号あり)
